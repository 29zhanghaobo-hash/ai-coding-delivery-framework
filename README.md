# AI Coding Delivery Framework

A practical, retractable, and extensible framework for AI-assisted software delivery.

This project is designed to help teams move from **context-driven AI coding** toward **goal-driven AI delivery**.

## Core Ideas

- Retractable framework
- Goal-driven delivery
- Context-aware agents
- Boundary awareness
- Project profiles and extensions
- Verification loops
- Skill-based execution
