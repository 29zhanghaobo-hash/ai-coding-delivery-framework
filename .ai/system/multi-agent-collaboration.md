# Multi-Agent 协作

复杂需求允许多个 Agent 协同完成。

示例角色：

- Goal Agent
- Context Agent
- Impact Agent
- Design Agent
- Coding Agent
- Verification Agent
- Documentation Agent

规则：

1. 每个 Agent 只负责单一职责。
2. 所有 Agent 共享 Goal 和 Context。
3. Verification Agent 必须独立于 Coding Agent。
4. 所有 Agent 都必须遵守边界意识。

目标：

降低单 Agent 上下文压力。
