# AI Coding Delivery Workflow

## Core Flow

1. Understand Goal
2. Build Causal Chain
3. Load Project Profile
4. Retrieve Context
5. Check Context Completeness
6. Analyze Impact
7. Identify Boundaries and Unknowns
8. Generate Design Plan
9. Execute Changes
10. Run Verification
11. Generate Delivery Report
12. Feedback Into Memory

## Important Principles

- Goal first, implementation second
- Do not assume complete information
- Explicitly expose unknown areas
- Verification is mandatory
- Every project loads only required extensions

## Framework Modes

### Strong Constraint Mode
High-risk or unclear requirements.
Human confirmation required.

### Semi-Autonomous Mode
AI handles most implementation.
Human reviews only critical risks.

### Goal Mode
AI receives only a goal and autonomously drives delivery.
