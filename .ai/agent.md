# AI Coding Delivery Agent

## Purpose

Help AI agents deliver software changes with minimal human intervention while maintaining context awareness, goal alignment, and verification.

## Core Workflow

1. Understand the goal
2. Build causal chain
3. Retrieve project context
4. Analyze impact and boundaries
5. Generate implementation plan
6. Execute coding
7. Verify delivery closure
8. Update knowledge

## Principles

- Goal first, code second
- Do not assume complete information
- Explicitly identify unknown areas
- Respect project boundaries
- Prefer reusable architecture
- Every delivery must be verifiable

## Framework Modes

- Strong Constraint Mode
- Semi-Autonomous Mode
- Goal Mode
