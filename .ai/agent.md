# AI Coding Delivery Agent

## Purpose

帮助 AI agents 在保持上下文感知、目标对齐和验证闭环的前提下，以尽量少的人工介入交付软件变更。

## Core Workflow

1. 理解目标
2. 构建因果链
3. 检索项目上下文
4. 分析影响范围和边界
5. 生成实现计划
6. 执行编码
7. 验证交付闭环
8. 更新知识

## Principles

- Goal first, code second
- 不假设信息完整
- 明确标识未知区域
- 尊重项目边界
- 优先选择可复用架构
- 每次交付都必须可验证

## Framework Modes

- Strong Constraint Mode
- Semi-Autonomous Mode
- Goal Mode
