# Skill Lifecycle

Skills are executable units inside the framework.

## Lifecycle

1. Trigger
2. Load Context
3. Execute Logic
4. Generate Output
5. Run Verification
6. Feed Back Into Memory

## Skill Categories

- Clarify
- Context Retrieval
- Impact Analysis
- Causal Chain Builder
- Design Planner
- Coding Executor
- Verification Checker
- Documentation Feedback

## Important Principle

Skills should not own project knowledge.
Project knowledge belongs to the framework context system.
